# 依存関係のタスクを強調表示する

画面上の要素が無いところで右クリック(iPad:タッチ&ホールド)し、コンテキストメニューから「依存関係のタスクを強調表示」、「非依存関係のタスクを透過表示」を選択します。いずれか、あるいは両方を適用することができます。
