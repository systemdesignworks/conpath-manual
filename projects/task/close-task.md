# タスクを完了済みにする

1. ツールバーの[選択]アイコンをクリックしてアクティブにします。
    
    [画像]
    
2. タスクをクリックして選択します。
    
    [画像]
    
3. 以下のいずれかの操作を行います。
    - 図形アクションの操作から[完了]アイコンをクリックします。
    - タスク右下の[編集]アイコンをクリックし、[完了済みにする]にチェックを入れます。
    - 右クリック(iPad:タッチ&ホールド)でコンテキストメニューを開き、[完了]を選択します。
    
    タスク、それに付随する次のリンクが完了表示になります。
    
    [画像]
